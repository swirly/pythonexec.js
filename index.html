Démonstrateur de pythonexec.js

Voici un démonstrateur de pythonexec, un petit outil en javascript basé sur les technologies existantes pour éxécuter Python dans le navigateur à partir de transpilage. Le code python est transformé en Javascript grâce la libraire Skulpt. Il est ensuite interprété par le navigateur.

L'ensemble est très léger. Une fois les quelques bibliothèques chargées, tout se passe dans le navigateur. Il n'y a donc pas de serveur à gérer, de charge serveur à anticiper.

Les blocs html

On peut utiliser des blocs prédéfinis qui enrichissent les balises html usuelles pour aller plus vite. Ici, nous sommes par exemple dans un bloc-cours qui s'utilise comme suit :

<bloc-cours> bloc de cours </bloc-cours>

Bien entendu, comme la bibliothèque highlight.js est disponible, on peut intégrer du code comme dans la ligne ci dessous

print("Hello world");

Signaler un danger

Un autre type de bloc est le bloc bloc-attention pour signaler un danger

Ne pas oublier les parenthèses autour de la chaîne de caractère.

Donner des consignes

Le bloc consigne permet aussi de donner des consignes. On le placera juste avant le bloc-python qui contient l'éditeur

  1. Ajouter au code suivant une ligne avec 
    print("Hello world");
  2. Éxécuter le code
 # Markdown ## Utilisation du markdown L'intégration de la bibliothèque javascript showdown permet de mettre des blocs de markdown * qui sont convertis à la volée * avec l'intégration transparente Celui ci est fait directement "en ligne"

Tous les blocs peuvent contenir des éléments, ou bien ils peuvent importer les élèments à partir d'un fichier en utilisant l'attribut data-file

Python

Le bloc python contient les boutons usuels :

  • Le bouton pour éxécuter le code
  • Un bouton pour recharger le code par défaut
  • Un bouton pour sauvegarder et un pour restaurer le code
  • Un dernier bouton pour le télécharger (pour éviter des copier/coller)

Et il contient aussi une sortie pour la console et pour les graphismes.

# Les lignes qui commencent par # sont des commentaires # Mettez ci dessous votre code
